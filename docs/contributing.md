# Contributing to Perth Traffic Watch

Thanks for your interest in contributing! This is an open-source, MIT-licensed project designed to be accessible to hobbyists, students, and professionals alike.

## Ways to Contribute

### 1. Hardware Improvements

- Enclosure design improvements (STL files in `hardware/enclosure/`)
- Power optimization suggestions
- Alternative component recommendations
- Field testing and durability reports

### 2. Firmware Development

- TensorFlow Lite Micro model optimization
- Power management improvements
- Cellular connectivity enhancements
- New feature implementation

### 3. Backend & Frontend

- API improvements
- Dashboard enhancements
- Data visualization
- Mobile app development

### 4. Documentation

- Deployment guides
- Tutorial videos
- Translation to other languages
- Case studies from your own deployments

### 5. Testing & Validation

- Accuracy testing of vehicle detection
- Power consumption measurements
- Weather resistance testing
- Long-term reliability data

## Getting Started

### 1. Fork the Repository

Click "Fork" on GitHub to create your own copy.

### 2. Clone Your Fork

```bash
git clone https://github.com/YOUR_USERNAME/perth-traffic-watch.git
cd perth-traffic-watch
```

### 3. Create a Branch

```bash
git checkout -b feature/your-feature-name
```

### 4. Make Your Changes

Follow the coding style of existing code. Add tests if applicable.

### 5. Submit a Pull Request

Push your branch and open a PR against the main repository.

## Coding Standards

### Firmware (C/C++)

- Use meaningful variable names
- Comment complex logic
- Follow existing formatting (2-space indent)
- Test on actual hardware before submitting

### Backend (Node.js/Python)

- Use ESLint/Black for formatting
- Add unit tests for new functions
- Document API endpoints

### Frontend (React/Next.js)

- Use TypeScript where possible
- Follow React best practices
- Ensure responsive design

## Hardware Guidelines

### Enclosure Design

- Use PETG or ASA for weather resistance
- Include cable glands for weatherproofing
- Ensure camera has clear view angle
- Consider mounting options

### Component Substitutions

If suggesting alternative components:
- Must be readily available
- Provide cost comparison
- Test before recommending

## Communication

### Issues

- Use GitHub Issues for bugs and feature requests
- Search existing issues first
- Provide clear reproduction steps for bugs

### Discussions

- Use GitHub Discussions for questions and ideas
- Be respectful and constructive
